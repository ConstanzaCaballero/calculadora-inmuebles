# Calculadora Inmuebles

## Qué es este proyecto
Script Python que calcula rentabilidad de inmuebles.
Pide: precio compra, alquiler mensual, gastos mensuales.
Calcula: rentabilidad bruta, neta y recomendación compra/no compra.

## Preferencias
- Código comentado en español
- Usar type hints
- Python 3.11

## Stack
- Python
- Flask (para la versión web)
